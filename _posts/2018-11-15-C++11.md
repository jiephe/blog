## std::atomic

对某些类型内部加了锁(如int,bool等)，以一个多线程互斥的int变量iVar为例，来看atomic的用法
~~~
#include <atomic>
#include <thread>

int count = 10000;
std::atomic_int iAtomicVar = count;
int iVar = count;

void fun1()
{
	for (int i = 0; i < count/2; i++)
	{
		iAtomicVar--;
		iVar--;
	}
}

void fun2()
{
	for (int i = 0; i < count/2; i++)
	{
		iAtomicVar--;
		iVar--;
	}
}

int main()
{
	std::thread t1(fun1);
	std::thread t2(fun2);

	t1.join();
	t2.join();

	printf("iAtomicVar: %d\n", iAtomicVar._My_val);
	printf("iVar: %d\n", iVar);

	while (true)
	{
		auto c = getchar();
		if (c == 'q')
			break;
	}

	return 0;
}
~~~

## std::shared_ptr

用法std::shared_ptr<class_type>，以std::thread为例 用法有以下几种

#### 第一种 先定义再初始化
~~~
std::shared_ptr<std::thread> td;
td.reset(new thread());
~~~
#### 第二种 通过赋值
~~~
std::shared_ptr<std::thread> td;
auto td1 = std::make_shared<std::thread>;
td = td1
//获取引用值
int a = td.use_count();
int b = td.use_count();
//获取指针值
std::thread* pThread = m_new_thread_.get();
~~~
